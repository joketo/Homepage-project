  • Heap comparison program

    This is a program to demonstrate the differences between different kinds of heaps. I programmed Fibonacci-, Binary-, and Binomi heaps. After that I tested on each heap how long it takes to insert, delete or take the min-value of them. These were tested on different values. The whole thing is made with Java. For details, see Github.

  • A tool for assigning shifts for home care workers

    A program I made with a pair as a database project. In home care there is x workers and y customers with 1 to 5 daily visits each. Our tool divides all the customers between the workers so that the workload per workers is as equal as possible. Some visits have special requirements for the worker which needs to be taken account. In this project, we used Python. For Finnish documentation and other details, see Github.
