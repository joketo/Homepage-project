Heap comparison program

This is a program to demonstrate the differences between different kinds of heaps. I implemented the Fibonacci, Binary, and Binomial heaps in Java and tested how long it takes to insert, delete or take the min value in them. These were tested with different values. For details, see Github.

A tool for assigning shifts for home care workers

A program I made with a pair as a database project. In home care there are a number of workers and customers which need to be visited 1 to 5 times each day. Our tool divides all the customers between the workers so that the workload per worker is as equal as possible. Some visits have special requirements for the worker which needs to be taken account. In this project, we used Python. For Finnish documentation and other details, see Github.
